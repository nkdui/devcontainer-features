## Feature Details

### Options

- **`conanVersion`** (string, default: "latest")  
  Version of Conan to install (e.g., "2.0.14", "1.60.2", or "latest")

- **`userHome`** (string, default: "/home/$_REMOTE_USER")  
  Home directory for the user (used for Conan cache)

### Supported Distributions

| Distro    | Test Scenario | Base Image         |
|-----------|---------------|-------------------|
| Ubuntu    | ubuntu        | ubuntu:latest     |
| Debian    | debian        | debian:latest     |
| Fedora    | fedora        | fedora:latest     |
| Arch Linux| archlinux     | archlinux:latest  |

## Installation Method

Conan is installed in a **global virtual environment** at `/opt/conan-venv`:

1. Checks if Python3 is installed (installs if missing)
2. Creates `/opt/conan-venv` virtual environment
3. Installs Conan via pip in the venv
4. Symlinks `/usr/local/bin/conan` → `/opt/conan-venv/bin/conan`
5. Sets permissions for all users

### Version Conflict Detection

The script checks for existing Conan installation:

- If `conanVersion: "latest"` → Accepts any installed version
- If installed version matches requested → Skips installation
- If versions mismatch → **ERROR** with conflict details

Example error:
```
ERROR: Version conflict detected!
  Requested version: 2.0.0
  Installed version: 1.66.0
Please remove the existing Conan installation or use 'latest' to accept any version.
```

## Testing Commands

### Validate metadata
```bash
devcontainer features validate --base-path ./src
```

### Run auto-generated test (Ubuntu with defaults)
```bash
devcontainer features test -f conan --skip-scenarios -i ubuntu:latest .
```

### Run all scenario tests
```bash
devcontainer features test -f conan --skip-autogenerated .
```

### Run specific scenario
```bash
# Test on Fedora
devcontainer features test -f conan -i fedora:latest .

# Test specific version scenario (conanVersion: 1.66)
devcontainer features test -f conan --filter specific_version --skip-autogenerated .
```

### Run everything (auto-generated + scenarios)
```bash
devcontainer features test -f conan .
```
